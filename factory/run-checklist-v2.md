# Per-Booklet Run Checklist

*Quick-scan companion to the station map. A checked box is not readiness —
readiness is the clarity behind the check.*

## Before the run
- [ ] Shared state is current (outline, voice framework, both conformance standards, editorial conventions)
- [ ] Skill files reflect all corrections logged from the previous booklet
- [ ] Model plan set: top tier for Stations 1–2 and 5 prompts; Sonnet for 3, 4, 6, 7; Haiku/Sonnet for 8

## The line
- [ ] **S1 — Brief drafted** → 🚪 **GATE: Brief approved by me**
- [ ] **S2 — Manuscript drafted** (full file) → 🚪 **GATE: full voice read passed**
- [ ] **S3 — Editorial conformance report clean** (or corrections applied as full replacement)
- [ ] **S4 — Antigravity relay complete** → all "revise" flags ruled on by me
- [ ] **S5 — Illustration prompts delivered** → 🚪 **GATE: images generated & approved by me**
- [ ] **S6 — Layout proof assembled**
- [ ] **S7 — Layout conformance report clean** against v2 standard
- [ ] **S8 — Payhip upload set assembled** (PDF, page copy, metadata, price)
- [ ] 🚪 **GATE: I click publish** (never delegated)

## After the run (the teaching loop — do not skip)
- [ ] Every human correction logged, tagged: one-off vs. missing rule
- [ ] Missing rules added to the relevant station's skill file **today**
- [ ] Promotion review: any station with N clean runs at its level? Promote one level.
- [ ] Note anything that should change in the station map itself

## Standing rules (apply at every station)
- Full file replacements only — never diffs
- No station skips its written standard, even when output "looks fine"
- If a station's output surprises you, the fix goes in the skill file, not just the file
