# Asset Placement Recommendation: Before the Build Starts

The `05-final-copy.md` manuscript for "Before the Build Starts" does not natively embed or reference the seven provided PNG assets. To preserve the integrity of the manuscript while still utilizing these helpful diagrams, layout designers should insert them at the following logical transition points or sidebars:

## Required Placements

**1. `asset-01-unclear-blueprint-construction-site.png`**
- **Recommended Placement:** Early in the booklet, near the introduction or Section 1, where the risk of starting without a clear plan is discussed.
- **Context:** Serves as a visual metaphor for the chaos of unguided building.

**2. `asset-02-wrong-order-vs-right-order.png`**
- **Recommended Placement:** Near the section outlining the correct sequential steps for AI agent planning.
- **Context:** Reinforces the core thesis of doing tasks in the correct order to save money and time.

**3. `asset-03-blueprint-stack-diagram.png`**
- **Recommended Placement:** In the section discussing the technology stack or required inputs.
- **Context:** Helps the reader visualize the layers required for the project.

**4. `asset-04-user-path-process-diagram.png`**
- **Recommended Placement:** Near any discussion of user flows or user journey mapping.
- **Context:** Shows how the user interacts with the final product.

**5. `asset-05-ai-guessing-zone-diagram.png`**
- **Recommended Placement:** Alongside the text explaining why AI fails when given ambiguous prompts.
- **Context:** Illustrates the gap between human intent and AI execution when details are missing.
